## About The Project
RESTful API with Node.js will make the client to server communication easier with HTTP requests. In this project, I have practiced how to develop a real-world, RESTful API with Node.js for a Blog System.

### Built With
The major technologies that i have used to build this project.
Here are:
* Node.js
* Express.js
* MongoDb


### Features

* Get all Posts
* Post CRUD
* Admin can manage own posts
* Stateless Authentication using JWT

**And so much more !!! Explore them by following "[Getting Started](#getting-started)" section.**

2. Run **npm install**
